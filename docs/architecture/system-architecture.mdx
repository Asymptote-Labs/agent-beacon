Asymptote supports two architecture paths: the open-source Agent Beacon architecture for supported local, CI, and cloud telemetry paths, and the managed Asymptote architecture for centralized visibility, governance, detections, and investigations.

## Choose An Architecture

| Architecture | Use when |
| --- | --- |
| Open Source Architecture | You want open-source telemetry for supported local, CI, and cloud agent surfaces, with customer-controlled records and forwarding. |
| Managed Architecture | You want managed ingest, retention, search, detections, fleet visibility, policy controls, identity mapping, approvals, SSO, RBAC, audit trails, and investigation workflows. |
