## Overview

Connect Okta from the Asymptote dashboard to automatically populate the users, groups, and devices tabs.

## Connect Okta

1. Log in to the Asymptote dashboard.
2. Go to **Connectors**.
3. Find the Okta integration.
4. Click **Connect**.
5. Enter your Okta org URL.
6. Enter your Okta API token.

<Frame caption="Use the Okta connector setup flow in the Asymptote dashboard to enter your Okta org URL and API token.">
  <img src="/images/asymptote_dashboard_okta_connector.png" alt="Asymptote dashboard Okta connector setup form." />
</Frame>

After the connector is authorized, Asymptote automatically syncs Okta data into the users, groups, and devices tabs in the dashboard.

## Generate an Okta API Token

Create the Okta API token from the Okta Admin Console:

1. Go to **Security**.
2. Open **API**.
3. Select **Tokens**.
4. Click **Create token**.

<Frame caption="Generate the API token from the Okta Admin Console before returning to the Asymptote connector setup.">
  <img src="/images/okta_token_generation.png" alt="Okta Admin Console API token generation screen." />
</Frame>

Copy the generated token and paste it into the Okta connector setup flow in Asymptote.
